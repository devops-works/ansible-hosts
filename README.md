ansible-hosts
=============

Populates /etc/hosts with hosts in inventory

Requirements
------------

None

Role Variables
--------------

- `hosts_more_entries` (default `[]`): additional hosts entries to add to hosts
  file
- `hosts_fix_cloud_config` (default: `true`): whetner to fix cloud config if
  present

Dependencies
------------

None
